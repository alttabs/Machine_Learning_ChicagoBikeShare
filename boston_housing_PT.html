### Implementao: Definir uma Mtrica de Desempenho
 difcil medir a qualidade de um modelo dado sem quantificar seu desempenho durante o treinamento e teste. Isso  geralmente feito utilizando algum tipo de mtrica de desempenho, atravs do clculo de algum tipo de erro, qualidade de ajuste, ou qualquer outra medida til. Para este projeto, voc ir calcular o [*coeficiente de determinao*](https://pt.wikipedia.org/wiki/R%C2%B2), R<sup>2</sup>, para quantificar o desempenho do seu modelo. O coeficiente de determinao  uma estatstica til no campo de anlise de regresso uma vez que descreve o quo "bom"  a capacidade do modelo em fazer estimativas. 

Os valores para R<sup>2</sup> tm um alcance de 0 a 1, que captura a porcentagem da correlao ao quadrado entre a estimativa e o valor atual da **varivel alvo**. Um modelo R<sup>2</sup> de valor 0 sempre falha ao estimar a varivel alvo, enquanto que um modelo R<sup>2</sup> de valor 1, estima perfeitamente a varivel alvo. Qualquer valor entre 0 e 1 indica qual a porcentagem da varivel alvo (ao utilizar o modelo) que pode ser explicada pelos **atributos**. *Um modelo pode dar tambm um R<sup>2</sup> negativo, que indica que o modelo no  melhor do que aquele que estima ingenuamente a mdia da varivel alvo.*

# TODO: Importar 'make_scorer', 'DecisionTreeRegressor' e 'GridSearchCV'
from sklearn.tree import DecisionTreeRegressor
from sklearn.model_selection import GridSearchCV
from sklearn.metrics import make_scorer



def fit_model(X, y):
    """ Desempenhar busca em matriz sobre o parmetro the 'max_depth' para uma 
        rvore de deciso de regresso treinada nos dados de entrada [X, y]. """
    
    # Gerar conjuntos de validao-cruzada para o treinamento de dados
    cv_sets = ShuffleSplit(n_splits=10, test_size = 0.20, random_state=0)

    # TODO: Gerar uma rvore de deciso de regresso de objeto
    regressor = DecisionTreeRegressor()

    # TODO: Gerar um dicionrio para o parmetro 'max_depth' com um alcance de 1 a 10
    params = {
        'max_depth': range(1,11)
        
    }
    
    

    # TODO: Transformar 'performance_metric' em uma funo de pontuao utilizando 'make_scorer' 
    scoring_fnc = make_scorer(performance_metric)

    # TODO: Gerar o objeto de busca em matriz
    #GridSearchCV(clf, parameters, scoring=scorer)
    grid = GridSearchCV(regressor,params,scoring=scoring_fnc, cv=cv_sets)

    # Ajustar o objeto de busca em matriz com os dados para calcular o modelo timo
    grid = grid.fit(X, y)

    # Devolver o modelo timo depois de realizar o ajuste dos dados
    return grid.best_estimator_
